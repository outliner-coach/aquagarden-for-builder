# UI 디자인 가이드

대상은 투명 오버레이 위에 떠 있는 **소형 제어 UI**(플로팅 버튼 + 확장 패널)뿐이다. 수조(3D) 자체가 주인공이므로 UI 크롬은 최대한 작고 조용해야 한다. (Tailwind 없음 — plain CSS, BEM 유사 클래스 사용)

## 디자인 원칙
1. **UI는 비켜선다.** 제어 패널은 작고 가장자리에 머문다. 수조와 바탕화면 시야를 가리지 않는다.
2. **도구처럼 보인다.** 위젯 컨트롤이지 마케팅 페이지가 아니다. 장식 요소 0개.
3. **조용한 모션.** 물고기/물의 움직임 외에 UI 자체가 번쩍이거나 튀지 않는다.

## AI 슬롭 안티패턴 — 하지 마라
| 금지 사항 | 이유 |
|-----------|------|
| 과한 backdrop-filter: blur() 글래스 효과 | glass morphism은 AI 템플릿의 가장 흔한 징후. (패널 가독성용 *약한* 반투명 배경은 허용, 단 멋부리기 금지) |
| gradient-text (배경 그라데이션 텍스트) | AI 슬롭 클리셰 |
| "Powered by AI" 류 배지 | 장식. 가치 없음 |
| box-shadow 네온 글로우 애니메이션 | AI 슬롭 |
| 보라/인디고 브랜드 색상 | "AI=보라색" 클리셰. 본 앱은 물색(청록) 포인트 |
| 모든 요소 동일한 rounded-2xl | 템플릿 느낌 |
| 배경 gradient orb (blur-3xl 원형) | 장식 |

## 색상
### 패널/버튼 표면 (반투명 다크)
| 용도 | 값 |
|------|------|
| 패널 배경 | rgba(15, 23, 28, 0.82) |
| 버튼 배경 | rgba(15, 23, 28, 0.7) |
| 보더 | rgba(255, 255, 255, 0.08) |

### 텍스트
| 용도 | 값 |
|------|------|
| 주 텍스트 | rgba(255,255,255,0.92) |
| 보조/라벨 | rgba(255,255,255,0.6) |
| 비활성 | rgba(255,255,255,0.35) |

### 포인트(물색)
| 용도 | 값 |
|------|------|
| 슬라이더 트랙/활성 | #4fd1c5 (청록) |
| 토글 ON | #4fd1c5 |
| 토글 OFF | rgba(255,255,255,0.2) |

## 컴포넌트
### 플로팅 버튼
```
40px 원형. background: 버튼 배경. border: 보더. 아이콘 SVG 1개(물방울/물고기).
cursor: grab (드래그 시 grabbing). hover 시 배경 +6% 불투명.
```
### 확장 패널
```
width ~220px. border-radius: 12px. padding: 14px 16px. background: 패널 배경.
보더 1px. 슬라이더/토글을 세로로 space-y 12px 배치. 표제는 작은 보조 텍스트.
```
### 슬라이더 (input[type=range] 커스텀)
```
트랙 height 4px, radius 2px. 채워진 부분 포인트색, 빈 부분 rgba(255,255,255,0.15).
thumb 14px 원형, 흰색.
```
### 토글 (click-through / hide)
```
스위치형. ON=포인트색, OFF=rgba(255,255,255,0.2). 라벨은 좌측 보조 텍스트.
```

## 레이아웃
- 제어 UI는 창의 한쪽 끝(기본 우측)에 위치. 접힌 상태는 플로팅 버튼만.
- 패널은 버튼 기준으로 펼쳐지며 수조 중앙을 가리지 않는다.
- 간격: 컨트롤 간 12px, 패널 안쪽 패딩 14~16px.

## 타이포그래피
| 용도 | 스타일 |
|------|--------|
| 패널 표제 | 12px / 600 / 보조 텍스트색 / letter-spacing 0.02em |
| 슬라이더 라벨 | 12px / 500 / 보조 텍스트색 |
| 수치 표시 | 12px / 600 / 주 텍스트색 (tabular-nums) |

## 애니메이션
- 패널 확장/접힘: 150ms ease-out (opacity + 약간의 translate/scale)만 허용.
- 그 외 UI 애니메이션 금지. (3D 씬의 움직임은 별개)

## 아이콘
- SVG 인라인, strokeWidth 1.5, currentColor.
- 아이콘을 둥근 배경 박스로 감싸지 않는다 (플로팅 버튼 자체가 컨테이너).
