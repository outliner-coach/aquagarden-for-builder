# PRD: 3D 디지털 아쿠아가든 (Aquagarden Overlay)

## 목표
업무 중 바탕화면 상단에 상시 떠 있는, 바탕화면이 투과되어 비치는 미니멀 3D 수족관 오버레이 위젯으로 시각적 휴식(힐링)을 제공한다.

## 사용자
PC(데스크톱/노트북)로 장시간 작업하며, 화면 한쪽에 가벼운 인터랙티브 배경 위젯을 켜두고 싶은 사용자.

## 핵심 기능
1. **상단 고정 와이드 오버레이 창** — 화면 상단에 가로로 긴 바 형태. 프레임 없는(frameless) 투명 창.
2. **Always on Top** — 다른 앱(브라우저, IDE, 문서) 위로 항상 노출. 기본 ON.
3. **Show/Hide 토글** — 버튼 한 번으로 수조 전체를 숨김/표시. 숨겨도 프로세스는 유지되고 제어용 플로팅 버튼은 남는다.
4. **Click-through(마우스 투과)** — '투과 활성화' 시 수조 영역 뒤의 바탕화면 아이콘·다른 창을 그대로 클릭/조작.
5. **알파 투명도** — 물·배경은 투명. 수조 너머로 바탕화면이 비친다.
6. **플로팅 버튼 + 확장 제어 패널** — 버튼 클릭 시 패널 확장. 버튼 드래그=창 전체 이동, 패널 드래그=패널만 이동.
7. **물고기 개체수 슬라이더** — 유영 물고기 마리 수를 실시간 증감(풀링 기반).
8. **조명 밝기 슬라이더** — 상단 3D 조명 세기 실시간 디밍(밤↔낮 분위기).
9. **미니멀 아쿠아스케이프** — 모래 바닥, 낮은 수초, 작은 바위. 물고기 움직임에 시선 집중.
10. **물고기 AI** — 자유 유영 + 소형 어류 군집(Boids) 행동.
11. **기포 파티클** — 바닥에서 수면으로 올라가는 가벼운 공기방울.
12. **백그라운드 성능 최적화** — 숨김 시 렌더 루프 정지, 풀링, 비동기 스폰으로 CPU/GPU 저점유.

## MVP 제외 사항
- 다중 모니터 정밀 배치/스냅, 창 크기 사용자 리사이즈 UI (초기엔 고정 와이드 바)
- 먹이주기·물고기 클릭 인터랙션 등 게임성 요소
- 물고기 종 도감/수집, 설정 영속화(파일 저장) 고도화 — MVP는 인메모리 상태 + 최소 영속화
- 사운드/BGM
- 사실적 물리 기반 물(굴절/반사) 셰이더 — MVP는 미니멀·경량 우선

## 디자인
- 다크/투명 기반. 수조 자체가 콘텐츠이므로 UI 크롬은 최소화.
- 제어 패널은 반투명 다크 무채색 + 포인트 컬러 1가지(물색 청록 계열). 글래스모피즘/네온 글로우 금지(UI_GUIDE 참조).
- 밤 분위기(저밝기, 은은한 발광) ↔ 낮 분위기(밝고 맑은 수조)를 슬라이더로 연속 전환. **기본값은 낮(밝은) 분위기**(레퍼런스 기준).

## 비주얼 레퍼런스 (`/reference_image.png`)
구현 목표 형태. 실행 시 이 이미지를 직접 보고 룩앤필을 맞춘다.
- **형태**: 화면 최상단 전폭에 걸친 가로 글래스 수조 바(높이 약 200~250px). 앞면 유리 패널의 미묘한 상/하단 엣지 하이라이트로 "유리 박스" 느낌을 준다.
- **투명**: 물·배경은 거의 투명해 수조 너머 바탕화면 아이콘/창이 그대로 비친다. 물을 파란 불투명 색으로 채우지 않는다 (옅은 틴트까지만 허용).
- **물고기**: 좌측에 소형어 밀집 군집(boids) + 전역에 다양한 개별어(네온테트라 청/적, 구피, 노랑·점박이 몰리, 엔젤피시 등). 색·크기·종 다양성이 핵심.
- **아쿠아스케이프**: 밝은 모래 바닥 + 낮은 초록 수초(잔디형, 곳곳에 군락) + 작은 자갈/바위 몇 개. 미니멀, 상단 시야는 비움.
- **라이팅**: 위에서 아래로 비추는 밝고 맑은 조명(낮). 수면 부근 하이라이트.
- **기포**: 전폭에 분산되어 수면으로 부드럽게 상승.
- **참고**: 이 이미지는 '수조 외관' 레퍼런스다. 제어 패널/플로팅 버튼 UI는 이미지에 없으며 UI_GUIDE를 따른다.
